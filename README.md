🚀 #Helplytics AI

AI-powered community support platform to connect learners with helpers.

🌍 ##Overview

Helplytics AI is a modern community-driven platform where users can ask for help, provide help, and grow together.

It solves a real-world problem where:

Students struggle to find quick help
Skilled individuals have no structured way to contribute

Helplytics bridges this gap using AI-powered suggestions and a smart request system.

✨ Features
🔐 Authentication (Login / Signup)
👤 User onboarding (skills, interests, role)
📊 Dashboard with insights
🔍 Explore help requests with filters
➕ Create and manage requests
📄 Request detail with helper actions
💬 Basic messaging system
🏆 Leaderboard & trust score
🔔 Notifications system
🤖 AI-powered suggestions (tags, categories, urgency)
🤖 AI Capabilities
Smart tag suggestions
Auto categorization
Basic urgency detection
Improved request writing suggestions
